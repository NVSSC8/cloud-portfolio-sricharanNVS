# Cloud & DevOps Engineer Portfolio

This repository contains a modern, responsive portfolio website for an aspiring Microsoft Azure Cloud & DevOps Engineer.

## What it does

- Fetches GitHub profile data and repositories
- Auto-updates project cards from GitHub
- Highlights Azure, Docker, Kubernetes, DevOps, and networking skills
- Designed for Azure Static Website Hosting

## Folder Structure

```text
portfolio/
├── index.html
└── assets/
    ├── css/styles.css
    └── js/script.js
```

## Run locally in VS Code

1. Open the folder in VS Code
2. Install Live Server
3. Right-click `index.html`
4. Choose **Open with Live Server**

## GitHub setup

1. Create a GitHub repository
2. Copy these files into the repository
3. Commit and push
4. Host the website with Azure Static Website Hosting or GitHub Pages

## Azure Static Website flow

Resource Group -> Storage Account -> Static website -> Upload files to $web -> Open endpoint
